# Security Policy

## Supported Versions

Only the latest release is supported with security patches. Always run the most recent tagged version.

## Reporting a Vulnerability

Report security vulnerabilities by opening a GitHub Issue at:

https://github.com/bzdvdn/maskchain/issues

For sensitive or critical vulnerabilities, please indicate "security" in the issue title and avoid including detailed exploit information in the initial public description. The maintainers will follow up to coordinate a private disclosure path.

## Response

We aim to acknowledge receipt of vulnerability reports within 48 hours and provide an initial assessment within 5 business days.
